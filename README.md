# EFP Everywhere Extension

This is a Chrome extension for viewing EFP card of Ethereum addresses and ENS names directly on web pages.

## Features

* Automatically detects selected ENS names (e.g., `.eth`) and Ethereum addresses (e.g., `0x...`) on the page
* Displays a full EFP card

## Development Steps

### 1. Install Dependencies

```bash
npm install
```

### 2. Build the Extension

```bash
npm run build
```

For development mode (auto-reload on file changes):

```bash
npm run dev
```

### 3. Load into Chrome

1. Open the Chrome browser
2. Go to `chrome://extensions/`
3. Enable **Developer mode** in the top-right corner
4. Click **Load unpacked**
5. Select the folder where the extension is located

### 4. Using the Extension

1. Visit any webpage containing Ethereum addresses or ENS names
2. Highlight an address or ENS name with your mouse
3. The EFP card will automatically appear showing detailed information

## Project Structure

```

├── webpack.config.js      # Webpack configuration

    ├── iframe-app.jsx     # Entry point for the React app
    ├── styles.css         # App styles
    └── components/

```
